# Smart Laundry Platform

A complete full-stack laundry service application featuring role-based dashboards (Admin, User, Provider), real-time order tracking, and automated workflow management.

## Features

- **User Dashboard:** Place orders, track status, and view history.
- **Service Provider Dashboard:** Accept jobs and update order status.
- **Admin Dashboard:** Manage users, providers, and view global statistics.
- **Real-time Updates:** Live status tracking for all users.

## Prerequisites

Before you begin, ensure you have met the following requirements:
- **Node.js**: You need Node.js installed on your machine. [Download Node.js](https://nodejs.org/)

## Installation & Setup

Follow these steps to run the project locally on your machine (Windows, Mac, or Linux/Ubuntu).

### 1. Download or Clone the Repository

You can download the ZIP file from GitHub or clone it using Git:

```bash
git clone https://github.com/babar123123/Smart-Laundry-Platform.git
```

### 2. Navigate to the Project Directory

Open your terminal (Command Prompt, PowerShell, or Terminal) and go to the project folder:

```bash
cd Smart-Laundry-Platform
```

### 3. Install Dependencies

Install all the required libraries and packages by running:

```bash
npm install
```

### 4. Run the Application

Start the server using the following command:

```bash
node server.js
```

You should see a message like:
> Server running on port 3000

### 5. Open in Browser

Open your web browser and visit:
`http://localhost:3000`

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (or local JSON storage depending on configuration)
